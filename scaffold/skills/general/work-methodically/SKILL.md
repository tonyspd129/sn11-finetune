---
name: work-methodically
description: General methodical problem-solving loop for terminal/agentic tasks.
always_on: true
---

Explore before you act: inspect files and run read-only commands to understand the current
state before changing anything. Make the smallest change that solves the task. Always re-run
the relevant check or command to confirm success before you finish. If a command fails, read
its error output carefully and address the specific cause rather than guessing.
